# ShopMart

This is a full-stack e-commerce web application where users can browse products, add to cart, and place orders. It also includes an admin panel to manage products and orders.

---

## Run Locally

**Prerequisites:** Node.js installed

### 1. Install dependencies

```bash
npm install
```

### 2. Start the app

```bash
npm run dev
```

App will run on:
👉 http://localhost:3000

---

## Features

* User authentication (login/register)
* Product listing & details
* Add to cart & wishlist
* Order placement
* Admin panel (manage products & orders)
* Razorpay payment integration
